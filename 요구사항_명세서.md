# SW_Bootcamp_13기 A반 2팀 — 요구사항 명세서

## 프로젝트 개요

- **팀명**: 보스 베이비 (A반 2팀)
- **팀원**: 정지운 · 박새란 · 김승우 · 주수원
- **한 줄 정의**: 사용자의 개인 데이터를 읽어 상황을 스스로 파악해 가전을 제어하며 정서적으로 교감하는 **TV 기반 IoT 비서**

### 핵심 차별점

1. **컨텍스트 기반 상황 인지** — 일정·위치·결제·날씨 등 여러 개인 데이터를 종합해 현재 상황을 판단한다.
2. **상황별 차등 동작** — 같은 “조용히 해줘”도 재택 발표 전과 퇴근 후에 다르게 동작한다.

### 시스템 동작 흐름

```
사용자 발화 → STT(노트북) → LLM API POST(노트북 → 라즈베리파이)
→ {text, homeSolution} 응답 생성(라즈베리파이)
→ TTS + 3D 캐릭터(노트북) → 가전 제어 / 결과 시각화
```

### 기술 스택

| 영역           | 구현                                                                                                  |
| -------------- | ----------------------------------------------------------------------------------------------------- |
| 분산 LLM       | 라즈베리파이 5(8GB) × 4 · Qwen3-30B-A3B q40 · distributed-llama · OpenAI 호환 `:9999` · 텐서 병렬(TP) |
| 음성 인식(STT) | Whisper                                                                                               |
| 음성 합성(TTS) | MeloTTS(KR) + OpenVoiceV2 음색 변환                                                                   |
| 프론트엔드     | React + Three.js · 3D 캐릭터(VRM)                                                                     |
| 가전 제어      | ThinQ 연동 → 시뮬레이션                                                                               |
| 개인 데이터    | 캘린더 · 위치 · 결제 · 날씨 · 유튜브 시청기록 등                                                      |
